Rotating Showcase | 05.09.2015

Target Audience:
Designed for a gaming site but can be used for anything really. The target audience is people from the ages of 15 to 50 male or female. Shows games for younger users to older users.

Client/User Needs:
No action to buy the items only view more detail. This is to pull users in to make them want to read more about the item on display. The item highlighted is shown on the main image the other items can be seen to the right of the main item.

Critique:
Item is in the center of the page and is bright and noticable. The list on the right makes it easier for the user to skim through the featured items and see what they are. Hovering over the other items changed the main image to the items image so the user can see it and if it interests them.

Offers and Deals Showcase | 05.09.2015

Target Audience:
Image above was designed for a gaming site but can be used for anysite really. The target audience in the image above is people from the ages of 15 to 50 male or female looking for deals or offers from the site.

Client/User Needs:
No action to buy the items only a images in a carousel and links to the right. The carousel has a group of 3 images at a time to show the user more offers at a time. The links to the right are to help the user find more offers and deals that are not on the carousel.

Critique:
The carousel is a lighter grey for the background so it dosn't blend into the main background. It also has arrows on the sides to show that it can show more. The list on the right has is titled so the user knows it has more links to other offers.
